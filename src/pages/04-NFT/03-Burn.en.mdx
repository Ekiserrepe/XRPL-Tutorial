# Burn NFT

On this page, learn how to Burn NFT!
You can burn NFs by using the NFTokenBurn transaction.

<LinkCard href='https://xrpl.org/nftokenburn.html' />

The following fields can be specified in an NFTokenBurn transaction:

- `NFTokenID`: (Required) Specify the ID of the NFT to be burned.
- `Owner`: Specifies the current NFT owner account when burning a `tfBurnable` NFT issued by yourself.
