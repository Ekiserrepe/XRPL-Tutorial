# Token Payment

このページでは、トークンの送金方法を学びましょう。
XRPの送金と同様にPaymentトランザクションを使用しトークンを送信することができます。

## 事前準備

トークンを保有するには事前にテスト用トークンのトラストラインを設定します。
このアカウント/トークンはテスト用にコミュニティから提供されており、トラストライン設定時にトークンを送信してくれます。

## トークンの送金

トークンを送信するPaymentトランザクションでは、`Amount`フィールドに送信するトークンの発行者(`issuer`)、通貨コード(`currency`)、送金量(`value`)を入力します。
