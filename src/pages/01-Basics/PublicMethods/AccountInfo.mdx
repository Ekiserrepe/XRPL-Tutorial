# アカウント情報の取得

`account_info`コマンドを使用することでアカウントの情報(`AccountRoot`オブジェクト)を取得することができます。

<LinkCard href='https://xrpl.org/ja/account_info.html' />

このコマンドにより、アカウントの残高や、アカウントの設定、アカウントの準備金のベースとなるオブジェクト数などのを取得することができます。

<AccountInfo/ >

## Example

import { Tab, Tabs } from 'nextra-theme-docs'

<Tabs items={['javascript', 'typescript']}>
  <Tab>

```js copy
const { Client } = require('xrpl')

const client = new Client('wss://testnet.xrpl-labs.com')

client.connect().then(() => {
  client
    .request({
      command: 'account_info',
      account: 'r9cZA1mLK5R5Am25ArfXFmqgNwjZgnfk59',
      validated: true,
    })
    .then((response) => {
      console.log(response)
    })
})
```

  </Tab>
  <Tab>

```ts copy
import { Client } from 'xrpl'

const client = new Client('wss://testnet.xrpl-labs.com')

await client.connect()
const response = await client.request({
  command: 'account_info',
  account: 'r9cZA1mLK5R5Am25ArfXFmqgNwjZgnfk59',
  validated: true,
})
console.log(response)
```
